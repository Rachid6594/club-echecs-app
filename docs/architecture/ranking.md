# Classement general - Sprint 08

## Regles de points

- Victoire : 3 points
- Match nul : 1 point
- Defaite : 0 point
- Abandon gagne : 3 points
- Abandon perdu : 0 point
- Timeout gagne : 3 points
- Timeout perdu : 0 point

Les matchs amicaux et les matchs de competition alimentent le classement general. Les classements competition utiliseront la meme logique de base avec un perimetre tournoi.

## Bonus competition

- Champion : +50 points
- Finaliste : +30 points
- Demi-finaliste : +15 points
- Participation validee : +5 points

## Tri

Le classement est trie par :

1. points totaux decroissants ;
2. victoires decroissantes ;
3. defaites croissantes ;
4. identifiant utilisateur.

## Rangs

Les seuils de rang sont disponibles via `rank_for_points`. Les badges visuels seront ajoutes au Sprint 09.
